# Solarium Energy - Solar Power Solutions

A modern, responsive website for Solarium Energy, showcasing solar energy solutions for residential and commercial clients. Built with HTML5, CSS3, and Vanilla JavaScript.

## Features

- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Interactive ROI Calculator**: Calculate your solar investment returns
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Performance Optimized**: Fast loading and optimized assets
- **Accessibility**: Built with web accessibility in mind

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript (ES6+)
- Google Fonts (Poppins)
- Font Awesome Icons (via CDN)

## Color Scheme

- Primary: Deep Navy Blue (#0b2239)
- Secondary: Solar Yellow (#ffb400)
- Background: White (#ffffff)
- Text: Dark Gray (#333333)
- Light Gray: (#f8f9fa)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/solarium-energy.git
   cd solarium-energy
   ```

2. Open `index.html` in your preferred web browser.
